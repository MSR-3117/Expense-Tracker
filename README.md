# Personal Expense Tracker (Java)

A console-based Java application for managing personal expenses using Object-Oriented Programming, the Collections Framework, and File Handling. The system supports adding, updating, deleting, viewing, and saving expenses with clean, modular code and persistent storage.

---

## Features

- Add new expenses  
- Remove existing expenses  
- Update expense details  
- View all recorded expenses  
- Calculate total expenditure  
- Save expense data to a file  
- Load saved data automatically at startup  
- Structured and maintainable codebase  

---

## Tech Stack

- Java (JDK 17 or above)  
- Object-Oriented Programming (OOP)  
- Collections Framework (ArrayList)  
- File Handling (BufferedReader, BufferedWriter)  
- Console-based user interface  

---

- **Expense.java** – Represents an expense record  
- **ExpenseManager.java** – Contains logic for expense operations  
- **FileHandler.java** – Manages file persistence for saving/loading data  
- **Main.java** – Console interface and program entry point  

---

## How to Run the Project

To run the project, ensure Java JDK 17 or above is installed on your system. Clone the repository using `git clone https://github.com/YOUR-USERNAME/expense-tracker.git` and navigate into the project directory with `cd expense-tracker`. Compile all Java source files inside the `src` folder using `javac src/*.java`, then run the application with `java src/Main`. The console-based interface will launch and any previously saved expenses will load automatically.


The application will launch with a menu-driven console interface and automatically load previously saved expenses.
